# 📚 Indian District Literacy Level Predictor

Predicts whether an Indian district is **Low / Medium / High** literacy
using only religion %, gender ratio, age groups, and population.
No literacy columns used — genuinely needs ML.

**Model:** Gradient Boosting Classifier · **Accuracy:** 73% · **640 districts · Census 2011**
